# PHP-Assignment-Repository

1. Task-1
In this task, I have created a registration form, which takes input from users i.e. Name, Phone Number, Email Id, Address with checkboxes and after submitting data, our data store in a "car_db" database.

2. Task-2
In this task, I have created a homepage of CarDekho website (not fully a clone website), but I have made a Dynamic website where admin can upload, delete, edit images & data in different sections, i.e.,
   i) Header
   ii) 1st Banner section
   iii) The most searched car section
   iv) Latest car section
   v) Footer

Then, these data stores and update in "cardekho_db" database.   

4. Database
I have export 2 database from phpMyAdmin (car_db, cardekho_db) in Database folder

5. Requirements :-
VSCode IDE
XAMPP Server (phpMyAdmin)
MySQL
PHP 8.4.10

Note - For connecting PHP with MySQL database, your password is also required in "config.php" file and "save.php" file
